# 🚀 Modern Responsive Portfolio Website

A professional, high-performance, and fully responsive portfolio website built using **HTML5**, **CSS3**, and **Bootstrap 5**. This project serves as a modern template for showcasing skills, experience, and projects with a focus on UI/UX and mobile-first design.

## 🌟 Key Features

- **📱 Fully Responsive**: Optimized for all screen sizes, including Mobile (375px), Tablet (768px), and Desktop (1440px+).
- **🎨 Premium UI/UX**: Includes modern design elements like glassmorphism, smooth gradients, and subtle micro-animations.
- **⚡ Performance Optimized**: Clean, modular CSS and minimal external dependencies for fast page load times.
- **🔍 SEO Best Practices**: Semantic HTML5 tags, optimized meta descriptions, and proper heading hierarchy.
- **🔝 Interactive Elements**: 
  - Sticky navigation with scroll-spy.
  - Smooth-scroll to sections.
  - Floating "Scroll to Top" button.
  - Dynamic hover effects on cards and buttons.

## 🛠️ Built With

- **HTML5**: For semantic structure.
- **CSS3 (Vanilla)**: For custom layouts, animations, and responsive media queries.
- **Bootstrap 5.3**: Used for the grid system and core UI components.
- **Bootstrap Icons**: For clean, scalable iconography.
- **Google Fonts**: "Work Sans" for modern typography.

## 📁 Project Structure

## 🚀 Getting Started

To run this project locally, simply clone the repository and open the `index.html` file in any modern web browser.

1. **Clone the repository:**
   ```bash
   git clone https://github.com/HussamKhaled01/My-portfolio.git
   ```

2. **Open the project:**
   Navigate to the project folder and open `index.html`.

## 🛠️ Customization

- **Colors**: Easily update the brand colors by modifying the CSS variables or global styles in `style.css`.
- **Content**: Update the sections in `index.html` to reflect your own projects and experience.
