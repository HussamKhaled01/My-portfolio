## 📖 Overview

A modern, high-performance portfolio website engineered to reflect a premium UI/UX standard. This project showcases my journey as a Full-Stack Developer, highlighting a strong foundation in **Software Engineering (GPA 87.3)** and a focus on enterprise-grade technologies like **ASP.NET Core**, **Angular**, and **SQL Server**.

> [!TIP]
> This portfolio is built with **zero heavy JS library dependencies**, ensuring maximum performance and a near-instant Time to Interactive (TTI).

- **Entity Framework Core**: Advanced ORM management and database-first/code-first migrations.
- **SQL Server**: Relational database design, T-SQL optimization, and stored procedures.

### 🔹 Frontend
- **Angular & TypeScript**: Creating dynamic, component-based SPAs with state management.

## 🌟 Key Features

- **🚀 Performance Optimized**: Custom-built carousel and timeline components using Vanilla JS to minimize bundle size.
- **📱 Fluid Responsiveness**: A mobile-first CSS architecture with dedicated breakpoints (`mobile.css`, `tablet.css`, `desktop.css`).
- **🔍 SEO & Accessibility**: Strict `h1 -> h2` heading hierarchy, ARIA-compliant attributes, and optimized meta tags for social sharing.
- **🎨 Interactive UX**: Smooth scroll-reveal animations, interactive modal system for project details, and a modern "Tinted Glass" aesthetic.
